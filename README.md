# zoodb-cli

ZooKeeper database utils

## Usage

```bash
# go run main.go import --help
NAME:
   zoodb-cli import

USAGE:
   zoodb-cli import [command options] [arguments...]

OPTIONS:
   --file value, -f value
   --prefix value, -p value
   --hosts value
   --session-timeout value   (default: 10s)
   --overwrite               (default: false)
   --clear, -C               (default: false)
   --help, -h                show help
```
